# BatteryChargeApp

This app automates house battery charging based on the status of a Wallbox EV charger.

## What It Does

- **Checks the status of your Wallbox charger**
- **Detects when the car is actively charging**
- **Sends a command to your Solis inverter** to start charging the house battery from the grid

When the car is not charging, the inverter is set to normal charging schedule.
